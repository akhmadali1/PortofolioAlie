Booking Room

Reservation for Room in Company

The project entails the development of a comprehensive room reservation system within a company, allowing users to conveniently book rooms for meetings or workspace purposes. Additionally, the system offers a valuable feature, as it provides real-time information about the availability of drivers. Users can access this information through a read-only Excel interface, enabling them to make informed decisions regarding transportation services. Furthermore, the project implements a Single Sign-On (SSO) solution, seamlessly connecting it to other applications within the company's ecosystem. This SSO integration streamlines user access and enhances security across various applications. The project adopts a modern technology stack with Next.js for the frontend, ensuring an interactive and responsive user experience, while the backend is powered by Golang.
